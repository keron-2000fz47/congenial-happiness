✨ Features

Everything you need to trade on pump.fun — out of the box.

🎯

Token Sniping

Detect and buy newly launched tokens on pump.fun in milliseconds via logsSubscribe or blockSubscribe.

📈

Take Profit / Stop Loss

Set percentage-based exit triggers. The bot automatically sells when your target profit or max loss is hit.

⚡

Geyser / gRPC Streaming

Connect via Yellowstone gRPC for ultra-low-latency real-time data. Jito ShredStream enabled by default.

🔁

Auto Retry Logic

Exponential backoff on 429 and RPC errors. Transactions are retried automatically for maximum reliability.

🪣

Rate Limiter

Token bucket rate limiter with burst support. Configurable max_rps prevents RPC bans.

📊

Bonding Curve Tracking

Monitor bonding curve fill progress in real time and predict token migration to PumpSwap AMM.

🔀

PumpSwap Migration Listener

Listen to graduation events when tokens move from bonding curves to the PumpSwap AMM pool.

⚙️

YAML Config

Each .yaml file in bots/ is an independent bot instance. Multi-bot support out of the box.

🏗️

Token Minting

Launch your own tokens on pump.fun directly from the bot. Full mint support included.
