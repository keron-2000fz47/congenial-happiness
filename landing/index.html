Everything You Need

Built for speed, reliability, and full control over your trading strategy.

🎯

Token Sniping

Detect and buy newly launched tokens on pump.fun in milliseconds using logsSubscribe or blockSubscribe.

📈

Take Profit / Stop Loss

Set automated exit conditions. The bot watches your positions and sells when your targets are hit.

⚡

Geyser gRPC Streaming

Real-time on-chain data via Yellowstone gRPC with Jito ShredStream enabled by default for ultra-low latency.

🔄

Dual Subscription

Both logsSubscribe and blockSubscribe are supported — choose the method that fits your RPC provider.

🪟

Built-in Rate Limiter

Token bucket algorithm keeps you within your provider's RPS limits. Automatic retry with exponential backoff on errors.

📊

Bonding Curve Tracker

Monitor the bonding curve fill progress and predict when a token will graduate to PumpSwap AMM.

🔀

PumpSwap Migration Listener

Automatically detect when tokens migrate from bonding curves to PumpSwap and react accordingly.

🏗

Token Minting

Launch your own tokens on pump.fun directly from the bot with full parameter control.

⚙

YAML Config Per Bot

Each .yaml file in bots/ is an independent bot instance. Run multiple strategies simultaneously.

Try the Bot

Connect your Solana wallet to access the live dashboard, configure your strategy, and start sniping tokens.

Wallet Connection Required
